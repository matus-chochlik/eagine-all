Detailed Description

template<typename Base = subscriber>
class eagine::msgbus::node_tracker< Base >

Service that consumes bus topology information and provides it via an API.

See also
service_composition

This class subscribes to the signals inherited from the node_tracker_base and tracks the information about the message bus topology, routers, bridges and endpoints, etc.
