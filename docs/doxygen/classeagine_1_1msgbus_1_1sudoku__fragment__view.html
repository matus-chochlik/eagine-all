void eagine::msgbus::sudoku_fragment_view< S >::for_each_cell ( Function  function) const

Calls the specified function for each cell in the fragment. The function should have the following arguments:

  • std::tuple<int, int> the fragment coordinate (in cell units),
  • std::tuple<int, int> the cell offset within the fragment,
  • basic_sudoku_glyph the glyph at the cell.
